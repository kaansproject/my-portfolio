# Personal Portfolio Website 

A modern, responsive, and lightweight 3-page personal portfolio website designed and developed from scratch to showcase my technical journey, skills, and projects in the IT and software infrastructure domains.

# 🛠️ Technologies Used
* **HTML:** Structured semantic markup for a clean, accessible layout.
* **CSS:** Custom styling, layout design, and responsive elements.

# 📄 Pages Included
1. **About Me:** A welcoming introduction showcasing my career focus, core technical interests, and professional journey.
2. **Experiences:** Detailed background regarding my computer programming education, vocational IT high school background, and my deep passion for tech support operations.
3. **Projects & Skills:** A dedicated section highlighting my technical stack, completed works, and active goals like CCNA & Azure certifications.
