# Personal Portfolio Website 🚀

A modern, responsive, and lightweight 3-page personal portfolio website designed and developed from scratch to showcase my technical journey, skills, and projects in the IT and software infrastructure domains.

# 🛠️ Technologies Used
* **HTML:** Structured semantic markup for a clean, accessible layout.
* **CSS:** Custom styling, layout design, and responsive elements.

# 📄 Pages Included
1. **Home (index.html):** A welcoming introduction showcasing my career focus and core technical interests.
2. **About Me:** Detailed background regarding my computer programming education and IT passion.
3. **Projects / Skills:** A dedicated section highlighting my technical stack and academic/professional works.
